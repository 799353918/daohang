# README

由于经常需要在不同的电脑不同的浏览器上操作，书签同步就很麻烦，所以想着写个导航页吧。

网站采用html+css+js+json 纯静态搭建，因为这么轻量的单页不想用框架不想用数据库。。

页面基础样式copy自<a target="_blank" href="https://github.com/geekape/geek-navigation">geekape</a>
